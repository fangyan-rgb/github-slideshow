#### 查找 hover 元素

打开开发者工具
打开 Sources 面板
执行用户操作让对象可见（例如鼠标悬停）
在元素可见的时候按下 F8（与“暂停脚本执行”按钮相同）
点击开发者工具左上角的“选取元素”按钮
点击页面上的元素

#### 字体强制换行和强制不换行

//不换行
.wrap {
white-space:nowrap;
}
//自动换行
.wrap {
word-wrap: break-word;
word-break: normal;
}
//强制换行
.wrap {
word-break:break-all;
}
